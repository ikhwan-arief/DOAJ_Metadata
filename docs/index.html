DOAJ Metadata Search

Publisher, journal, and article search.

Search by journal title, publisher name, article title, or topic keywords related to the article you want to find. The application uses live DOAJ search results in the browser and supports Boolean query logic such as AND, OR, NOT, parentheses, and exact phrases in quotation marks.

Results are grouped into publishers, journals, and articles. Select any result for detail.

Journal matching for manuscript abstracts.

Paste an English abstract, then find DOAJ journals whose scope, subjects, and article themes most closely align with the manuscript.

Journal Matching uses live DOAJ API results only. The submitted abstract stays in the browser.

DOAJ journal statistics.

Explore global DOAJ journal indicators, filters, charts, maps, and table views built from the DOAJ API and refreshed every 3 hours by GitHub Actions.

DOAJ inclusion pre-check.

Use this tool to review your journal website before applying to the Directory of Open Access Journals (DOAJ). The result is indicative only: DOAJ will always make its own independent decision.
